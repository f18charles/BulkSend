# BulkSend 📱

BulkSend is a secure, elegant, 100% free browser-based companion application that allows you to orchestration and launch personalized SMS campaigns directly using your mobile phone's built-in carrier SIM card.

## 🚀 How It Works
Rather than using expensive, third-party centralized API gateways (which charge per-character fees and expose contact lists to privacy issues), **BulkSend** works fully in-browser through native system-level deep linking:

1. **Upload or Paste Contacts**: Paste row data directly from Excel, Google Sheets, or CSV tables (containing names and phone numbers).
2. **Design a Personalized Template**: Use tags like `{Name}` and `{Phone}` inside your custom template.
3. **Scan QR Code**: Open the exact campaign setup on your phone instantly via a secure QR code—no databases are involved (all data is securely packed directly in the URL).
4. **Instantly Dispatch**: Click the actions to pre-fill your phone's native message app. Press send inside your phone! All SMS fees are covered strictly by your existing mobile subscription plan.

## ✨ Features
- **Spreadsheet/CSV Autoguess**: Copy-paste raw rows and columns; BulkSend parses telephone formats and name cells automatically.
- **Dynamic Tag Personalization**: Custom variables replace tags dynamically on dispatch.
- **Deep SMS Integration**: Triggers carrier native SMS views on iOS and Android devices dynamically.
- **QR Campaign Transfer**: Instantly send contacts from computer screen to your handset using purely offline client-side URL encoding.
- **Strict Privacy**: Your private contact lists and text bodies never touch a server database.
